# Sistema de Suporte ao Usuário

Plataforma de suporte acessível via **web** e **aplicativo móvel**, criada para oferecer uma experiência unificada entre clientes e equipes de atendimento.

## Funcionalidades

- Abertura e acompanhamento de chamados
- Notificações em tempo real
- Base de conhecimento com artigos e tutoriais
- Chat integrado para suporte rápido
- Dashboard para gestão de tickets e relatórios

## Tecnologias

- Front-end: Angular / Mobile
- Back-end: NestJS / Java
- Banco de Dados: PostgreSQL

## Objetivo

Centralizar o atendimento, reduzir tempo de resposta e melhorar a experiência do usuário final.
